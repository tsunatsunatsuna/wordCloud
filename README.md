# ワードクラウド生成アプリ

日本語テキストからワードクラウドを生成するWebアプリです。
Kuromoji.jsを使用して名詞を抽出し、wordcloud2.jsで可視化します。

## 機能

- テキスト入力またはファイルアップロード
- ストップワード（除外単語）指定
- カスタマイズ可能なワードクラウド生成
- PNG形式でのダウンロード
- プリセット機能（設定の保存・読み込み）

## 使い方

1. テキストを入力またはファイルをアップロード
2. 設定を調整（プリセットを選択してもOK）
3. 「ワードクラウドを生成」ボタンをクリック
4. 生成されたワードクラウドを「PNGでダウンロード」ボタンで保存

## 開発環境

- HTML/CSS/JavaScript
- Kuromoji.js（形態素解析）
- wordcloud2.js（ワードクラウド描画）

## デプロイ

GitHub PagesやVercelなど、静的ファイルホスティングサービスにアップロードしてご利用ください。
